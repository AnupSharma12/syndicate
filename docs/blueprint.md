# **App Name**: Syndicate ESP

## Core Features:

- Tournament Registration: Allow users to register and pay for upcoming esports tournaments.
- Event Schedule: Display a detailed, filterable event schedule for tournaments and events.
- Game Filtering: Enable filtering events by specific games or esport titles.

## Style Guidelines:

- Use a dark background to create a sleek and modern feel.
- Primary color: Red to convey energy and excitement, with the goal of harmonizing visually with a dark background.
- Secondary color: White, used for highlights and interactive elements to maintain visual consistency.
- Accent color: Blue, used for highlights and interactive elements to maintain visual consistency.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and shorter text segments.
- Body font: 'Inter' (sans-serif) for body text to ensure readability.
- Use the dragon logo, with red, white and blue accents, to represent the brand.
- Implement a grid-based layout to ensure consistency and responsiveness across different devices.
- Incorporate subtle, cool scrolling animations to enhance user experience. Loading animations should include a dragon.
